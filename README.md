# 📝 To-Do List (Windows Forms)

## 📌 Project Description
A simple To-Do List application built with **Windows Forms**.  
It provides the following basic features:
- Add a new task to the list.
- Delete an existing task.
- Restart the program to clear all tasks and start fresh.

## ⚙️ Requirements
- Visual Studio (2019 or later).
- .NET Framework (4.7.2 or later recommended).

## 🚀 How to Run
1. Open the project in Visual Studio.
2. Build the solution.
3. Run the application using the **Start** button.

## 🛠️ Usage
- Enter a task in the input box and click **Add**.
- Select a task from the list and click **Delete** to remove it.
- Click **Restart** to clear all tasks and restart the program.

## 📂 Project Structure
- `Form1.cs` : Contains the UI and main logic.
- `Program.cs` : Entry point of the application.
- `bin/` and `obj/` : Build output folders (excluded from Git).

## 📄 Notes
- This project is for learning and practice purposes.
- Future improvements could include saving tasks to a file or database.
